# PairUp

Pair Up is a JavaScript project that randomly matches student names with teacher names and displays the assignments in a simple interactive table. It demonstrates basic front end logic using arrays, randomization, and DOM manipulation, making it a clear and beginner friendly example of practical JavaScript usage.

# Features

Random assignment of students to teachers
Simple interactive table display
Built with core JavaScript without frameworks
Beginner friendly logic and structure

# Technologies Used

JavaScript,HTML5, CSS3,

# Project Purpose

# How it works

When the Assign button is clicked, students are paired randomly with teachers, and the results are shown in a table. This provides a basic example of dynamic UI updates using JavaScript.
